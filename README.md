# 🌑 HORROR MINI - ALPHA 0.03
**Geliştirici:** Farmant Studios
**Platform:** PC (Python/Ursina)

## 🎮 Hakkında
Voxel tabanlı, dinamik dünya üretimine sahip bir korku-survival deneyimi. Dağların arasında pusu kuran bir şeyler var...

- **WASD:** Hareket
- **Space:** Zıplama / Oyunu Başlat
- **Sol Tık:** Blok Kırma
- **K Tuşu:** Blok Koyma (Taş)
